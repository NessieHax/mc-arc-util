# ARC2ZIP - ZIP2ARC

Simple scripts to convert Minecraft Legacy Console .arc files to .zip files and back to .arc .

## Usage

```cmd
arc2zip.py <Media.arc>
```
Writes to `arc.zip`\
_should support drag n' drop_

---

```cmd
zip2arc.py <*.zip>
```
Writes to `out.arc`\
_should support drag n' drop_
